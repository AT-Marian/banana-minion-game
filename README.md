# Bananaaa - The Fun Math Puzzle Game

This is a  web based math puzzle game where players identify the number represented by a banana in math equations using a Banana API.

# Technologies
- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS + Shadcn/UI
- Database: Supabase (PostgreSQL)
- Authentication: Supabase Auth
- Real-time: Supabase Realtime

# Features
- Single Player mode with 5 rounds and 60 second timer
- Multiplayer mode with room codes and real-time sync
- Leaderboard showing top 20 players
- Coin rewards and sticker system
- Secure answer checking via Supabase Edge Functions
